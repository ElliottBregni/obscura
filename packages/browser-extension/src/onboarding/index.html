1 · install the native host

The extension talks to obscura through a native messaging host — a tiny shell script that spawns the agent runtime. Install it once with either of these: 

cd $OBSCURA_REPO
make ext-install
obscura-browser install
cd $OBSCURA_REPO/packages/browser-extension/native-host
./install.sh

Each writes com.obscura.host.json into every Chrome-family browser dir on the machine, kills any stale host process, and prints the resolved python path.

2 · (optional) inject backend secrets

Chrome launches the host with an empty environment. For Copilot the launcher auto-reads gh auth token. For Anthropic, OpenAI, etc., drop an env file: 

# ~/.obscura/browser-host.env
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...

3 · open the side panel

Pin the ● obscura icon, then click it — or open Chrome's side-panel launcher on the right of the address bar.

chrome://extensions
