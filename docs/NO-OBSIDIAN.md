# Using FV-Copilot Without Obsidian

The vault is just a folder with markdown files and symlinks. **Obsidian is optional!**

## Why You Don't Need Obsidian

- Vault = `~/FV-Copilot/` directory
- Context files = `.md` markdown files
- Skills = organized in folders
- Symlinks work in any editor

## Using Your Preferred Editor

### VS Code
```bash
cd ~/FV-Copilot
code .
```
- Install Markdown Preview extensions
- Use file explorer to navigate
- Edit `.md` files like any code

### Vim/Neovim
```bash
cd ~/FV-Copilot
vim README.md
```
- Use `:Ex` or NERDTree for navigation
- Markdown syntax highlighting works out of box

### Any Editor
The vault is just files and folders. Use whatever you prefer!

## What Obsidian Adds (If You Want It)

### Visual Navigation
- File browser with folder tree
- Quick search across all notes
- Backlinks panel

### Knowledge Graph
- See connections between files
- Visualize repo relationships
- Find related context

### Markdown Enhancements
- Live preview
- Embedded images
- Callout blocks

## Recommendation

**Start without Obsidian** if you prefer your current workflow.

**Try Obsidian later** if you want:
- Visual knowledge management
- Backlinks between notes
- Graph view of connections

The scripts work the same either way! 🎯
