sequenceDiagram

    Scheduler->>Node: Schedule on matching node
    Node->>Monitor: Start container with privileged mode
    Monitor->>Monitor: Load TLS certificates
    Monitor->>Monitor: Set environment variables
    Monitor->>Monitor: Execute entrypoint.sh
    Monitor->>Neuron: Connect to Neuron runtime
    loop Every 5 seconds
        Neuron->>Monitor: neuroncore_counters
        Neuron->>Monitor: memory_used
        Neuron->>Monitor: execution_stats

    Metrics-->>K8s: Metrics available via HTTPS
