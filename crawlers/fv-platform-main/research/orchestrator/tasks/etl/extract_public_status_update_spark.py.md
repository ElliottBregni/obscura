    Main->>S3: Try read latest parquet
    alt Latest exists
        S3-->>Main: Return df_latest
        Main->>Main: Get max_id from df_latest
        Main->>Main: Filter new records (id > max_id)
        Main->>S3: Write filtered data to versioned path
        Main->>S3: Read back versioned data
        Main->>S3: Append to latest path
    else Latest does not exist
        Main->>S3: Write full data to versioned path
        Main->>S3: Read back versioned data
        Main->>S3: Write to latest path
    end
    Main->>S3: Read latest path for deduplication
    Main->>Main: Apply window function and ranking
    Main->>S3: Write to duplicate temp path

    Main->>S3: Write final deduplicated data to latest path
