```mermaid
sequenceDiagram
    participant AWS_Event as AWS Event (SQS/SNS Records)
    participant Lambda as lambda_handler
    participant DB as FvDatabaseConnector (DB_CONN)
    participant Cursor as DB Cursor
    participant Request as RequestHandler
    participant Carbon as carbon_care module
    participant FV_Lambdas as fv.aws.lambdas

    AWS_Event->>Lambda: Invoke with event
    Lambda->>Lambda: log records
    Lambda->>DB: DB_CONN.establish_connection()
    DB->>Cursor: DB_CONN.get_cursor()
    loop for each record
        Lambda->>Request: RequestHandler(record)
        Request->>Request: parse_request()
        Request->>FV_Lambdas: get_path_parameter(...)
        Request->>Request: validate_parameters()
        alt shipment_id invalid
            Request->>Request: handle_validation_error()
            Request-->>Lambda: raise BadRequestError
        else valid
            Request->>Carbon: get_co2_data(cursor, shipment_id, is_shipment_completed)
            alt results present
                Carbon->>Carbon: iterate shipments
                Carbon->>Carbon: save_co2_data(cursor, shipment)
            end
            Request->>Lambda: format_results()
        end
    end
    Lambda->>FV_Lambdas: make_response([], 200)
    FV_Lambdas-->>Lambda: HTTP response
    Lambda-->>AWS_Event: complete
```
